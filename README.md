# UTM Timetable Web Application
##### This project is a web application built using Vue 3 to display timetable and academic-related information for Universiti Teknologi Malaysia (UTM). The main objective of this project is to practice API integration, state management, and frontend development using modern web technologies.
---
## Overview
This application consumes an API provided by the lecturer and presents structured information such as:
- Student timetable
- Academic schedule
- Related university information
The focus of this project is to understand how to manage API data efficiently and display it in a clean, user-friendly interface.
---
## Tech Stack
- Vue 3 (Composition API)
- Vite
- Vue Router
- Pinia (state management)
- Axios (API handling)
- Tailwind CSS (styling)
- Element Plus (UI components)
- Vue Cal (calendar view)
- Vue3 Datepicker
---
## Features
- Fetch and display timetable data from API  
- Calendar-based schedule visualization  
- Responsive and clean UI design  
- State management using Pinia  
- Modular and scalable Vue structure  
---
## Project Setup
Install dependencies:
```sh
npm install
```
Run development server:
```sh
npm run dev
```
Build for production:
```sh
npm run build
```
